# 学習用アナログ時計タイマー

目標時間を設定し、時計の針が進むのを見ながら学習できるタイマーです。

## 使い方

1. `index.html` をブラウザで開く
2. 目標時間（分）を入力して「目標設定」をクリック
3. 時間の進み方（等倍・1.5倍速・2倍速）を選択
4. 「スタート」でタイマー開始

## ファイル

- `index.html` - 単体で動作するHTML（CSS・JavaScript含む）

## 動作環境

現代のブラウザ（Chrome, Firefox, Edge, Safari など）でそのまま開いて利用できます。
